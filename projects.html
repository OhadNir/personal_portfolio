Title:

Connect 4 AI

Description:

This is my final project in Introduction to Cognitive Science. My goal is to make an Artificial Intelligence that can make decisions based on the players’ moves to optimize their chances of winning. The decision algorithm I'm implementing is called Minimax and to make it faster, I'm using a supporting algorithm called Alpha-beta pruning, which helps discard unnecessary decisions. The only programming language used is C++. I also used GDB and Dr. Memory for debugging the program and identifying memory leaks.
